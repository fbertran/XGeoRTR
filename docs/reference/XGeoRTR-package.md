# XGeoRTR: Backend-Neutral Explainable Geometry State

Backend-level tools for standardizing explanation geometries into
backend-neutral `xgeo_state` objects with embeddings, diagnostics,
selection state, multiscale level-of-detail summaries, and selected
long-table views for downstream use-case packages. Rendering is
delegated to downstream frontends such as ggWebGL. The frozen
XGeoRTR/ggWebGL boundary is documented in `INTERFACE_FREEZE.md`.
