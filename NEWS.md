# XGeoRTR 0.1.0

- Initial platform package split from `shapViz3D`.
- Added the neutral `xgeo_data` object model for spatial explanation tables.
- Added `xgeo_scene()`, `geom_xgeo_surface()`, `render_webgl()`, and `animate_camera_orbit()`.
- Added a generic spatial demo dataset, example script, tests, and a getting-started vignette.
